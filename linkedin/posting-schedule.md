## How to schedule natively on LinkedIn (~2 min per post)

1. Start a post → paste the copy from the week file.
2. For carousels: add document → upload the PDF exported from the Canva design.
3. Click the clock icon next to Post → set the date and time above → Schedule.
4. LinkedIn allows scheduling up to 3 months ahead, so all 10 can be queued in one sitting (~20 min).
